📦 安装指南

⏱️ 15 分钟 | 难度 ⭐ | 零基础也能看懂的安装教程

本教程将带你从零开始，15 分钟快速部署 OpenClaw 环境。包含 Node.js 安装、依赖配置、渠道接入等完整步骤。

🎯 前置要求

  • Node.js 18+ 环境
  • Git 版本控制工具
  • 一个 IM 账号（微信/Telegram/Discord 等）

📝 步骤 1: 克隆项目

# 克隆 OpenClaw 仓库
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 安装依赖
npm install

预期输出:

added 1234 packages in 45s

⚙️ 步骤 2: 基础配置

# 复制示例配置
cp config.example.json config.json

编辑 config.json：

{
  "gateway": {
    "port": 3000,
    "host": "localhost"
  },
  "channels": {
    "telegram": {
      "enabled": false,
      "botToken": "YOUR_BOT_TOKEN"
    }
  },
  "models": {
    "default": "qwen3.5-plus"
  }
}

🔗 步骤 3: 渠道接入

Telegram

  1. 联系 @BotFather 创建机器人
  2. 获取 Bot Token
  3. 填入配置文件的 telegram.botToken
  4. 设置 telegram.enabled: true

微信

  1. 使用 BlueBubbles 或 WeChat 协议库
  2. 扫描二维码登录
  3. 配置 Webhook 地址

🚀 步骤 4: 启动服务

# 启动 Gateway
npm start

# 或使用开发模式
npm run dev

预期输出:

✅ Gateway started on port 3000
✅ Channels loaded: telegram, wechat
✅ Ready to receive messages!

✅ 验证安装

发送消息到你的机器人，应该能看到回复！

❓ 常见问题

npm install 失败

尝试使用淘宝镜像：

npm config set registry https://registry.npmmirror.com
npm install

端口被占用

修改配置文件中的 gateway.port 为其他端口

渠道无法连接

检查网络环境和 Token 配置，参考对应渠道的接入文档
